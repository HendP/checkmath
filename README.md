# Checkmath

Game kuis dengan tema matematika tingkat SD, SMP dan SMA pada platform desktop yang dibuat pada tahun 2017 sebagai tugas kuliah pemilik repository ini dan tim.

# System Requirement

- IDE Netbeans 8.2
- JDK 8
